# Security Policy

## Supported Versions

We currently support the following versions of the Maesn MCP server with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you want to report a security vulnerability, please contact us via contact@maesn.com.

We aim to respond to all inquiries within 24 hours. For accepted vulnerabilities we will provide an ETA for an updated version.
